# Responsive Sidebar Menu

Projeto desenvolvido com HTML e CSS que cria uma sidebar moderna e responsiva. O menu lateral pode ser aberto e fechado através de um botão interativo utilizando apenas HTML e CSS, sem necessidade de JavaScript.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Font Awesome

## ✨ Funcionalidades

- Sidebar responsiva
- Menu lateral animado
- Botão abrir/fechar menu
- Navegação com ícones
- Redes sociais integradas
- Checkbox Hack
- Sem JavaScript

## 📂 Estrutura do Projeto

```bash
Responsive-Sidebar-Menu/
│
├── index.html
│
├── css/
│   └── style.css
│
├── img/

│
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.
